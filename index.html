Waa'ee Koo & Seenaa Ko

Jireenyi barumsa dha. Abbaa koo baruumsaaf fedha adda qabu biratti guddachuun koo obsa fi jabeenya na barsiiseera. Barataa Midwifery ta'ee, gama kanaan Management fi Coding barachuun koo immoo mul'ata koo bal'iseera.

"Barumsi meeshaa humna qabu kan ati addunyaa ittiin jijjiiruuf fayyadamtu dha." — Nelson Mandela

Management: Ogummaa Bulchiinsaa

Bulchiinsi gaariin utubaa milkaa'inaati. Karoorsuu, qindeessuu fi qabeenya jiru sirriitti fayyadamanii galma gahuun ogummaa kooti. Hojiin hundi ijaarsa cimaa gaafata.

"Management jechuun wantoota sirriitti hojjechuudha; Leadership jechuun immoo wanta sirrii ta'e hojjechuudha." — Peter Drucker

Midwifery: Ogummaa Yaalaa

Jireenya haaraa addunyaatti fiduun eebba dha. Fayyaa haadhaa fi daa'imaa eeguun ogummaa qulqulluu qofa osoo hin taane, jaalala fi garaa laafummaa gaafata. Hawaasa kootiif tajaajila qulqulluu kennuun boonummaa kooti.

"Iddoo jaalalli jiru, dinqiin ni uumama." — Florence Nightingale

Web Development: Addunyaa Koodii

Koodiin afaan addunyaa isa haaraati. HTML fi CSS fayyadamee waan sammuu koo keessa jiru gara addunyaatti fiduun dandeettii kalaqaa koo naaf mirkaneessa. Teeknoolojiin meeshaa jireenya keenya ittiin salphisnu dha.

"Namni hundi koodii barreessuu barachuu qaba, sababni isaas akkamitti akka yaadan si barsiisa." — Steve Jobs
